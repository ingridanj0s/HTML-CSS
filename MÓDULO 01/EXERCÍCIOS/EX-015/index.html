Capítulo 1.1

Aplicar o estilo externo significa trazer o style para o seu site a partir de um arquivo externo, que alterará o estilo de todas as páginas, sem que seja necessário copiar e colar várias vezes.
Inclusive é possível adicionar mais de um link na área head, quando eu quero um estilo específico somente para algumas páginas, por exemplo.

No mesmo arquivo podemos sim utilizar CSS inline, CSS interno/local e CSS externo.
