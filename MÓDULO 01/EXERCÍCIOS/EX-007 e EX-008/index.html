Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântica = não tem significado, somente formato).
Já nesta frase, temos um termo em destaque contendo a nova semântica do HTML5, utilizando a tag STRONG.

Itálico

Nessa frase temos um termo em itálico usando a tag I (não semântica = não tem significado, somente formato).
Já nesta outra, temos outro termo em destaque usando a tag EM (semântica do HTML5).

Texto marcado (marca texto)

Podemos criar também um texto marcado usando a tag MARK + style para utilizar o CSS e mudar a cor do marcador, por exemplo.

E se eu tiver mais de uma frase, um texto ou termo que eu queira marcar - como este - será necessário abrir a tag MARK dentro do Style (lá em cima) ao invés de uma por uma.

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo (sendo que a tag BIG já está obsoleta, portanto, fica em vermelho, enquanto a tag SMALL ainda está funcionando).

Texto deletado

Podemos criar um texto como "excluído" para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos também marcar um texto como "inserido" para dar ênfase e indicar que ele foi adicionado posteriormente.

Lembrando que ainda temos a tag U que sublinha, mas não deve ser utilizada mais para isso, uma vez que temos a tag INS (que é semântica, conforme o HTML5).
