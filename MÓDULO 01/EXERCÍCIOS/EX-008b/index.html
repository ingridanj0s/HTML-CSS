Código-fonte

O comando document.getelementById('teste') é um exemplo de comando em linguagem JavaScript. Para destacar com a fonte mono-espaçada, utilizamos a tag CODE.

Para escrever um código dentro do texto, usando a identação, será necessário adicionar tudo dentro da tag PRE utilizando também a tag CODE.
