Usando links externos (links que apontam para outro site)

Você pode acessar meu repositório público no GitHub clicando aqui.

Você também pode acessar o meu LinkedIn.

Usamos a tag <a> para incluir o link da página para onde queremos direcionar o usuário e incluímos o atributo target="_blank" para que o link seja aberto em uma nova aba ou target="_self" para que o link seja aberto em cima da aba atual (sobrepondo).
